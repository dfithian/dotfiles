# Setup

1. Install [brew](https://brew.sh/)
2. `brew install zsh`
3. `sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`
4. Install dotfiles: `./make.sh`
